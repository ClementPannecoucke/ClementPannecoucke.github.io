Intelligence Artificielle

Python TensorFlow Computer Vision CNN Deep Learning

Description du projet

Ce projet d'intelligence artificielle est un système de reconnaissance d'images qui utilise des réseaux de neurones convolutifs (CNN) pour classifier des objets dans des images. Il a été développé avec Python et TensorFlow.

Le modèle a été entraîné sur un large ensemble de données d'images étiquetées et peut reconnaître plus de 100 catégories d'objets différents avec une précision élevée.

Fonctionnalités principales

  • Classification d'images en temps réel
  • Interface utilisateur simple pour télécharger et analyser des images
  • Visualisation des couches du réseau de neurones
  • Analyse des performances du modèle
  • API REST pour l'intégration avec d'autres applications

Méthodologie et résultats

J'ai utilisé une architecture de réseau de neurones convolutifs inspirée de ResNet, avec plusieurs couches de convolution suivies de couches de pooling et de normalisation par lots. Le modèle final a été entraîné pendant 50 époques sur un GPU NVIDIA RTX 3080.

Le modèle a atteint une précision de 94,7% sur l'ensemble de test, ce qui est comparable aux performances des modèles état de l'art pour cette tâche.
